## Getting started

To install `@candy-doc/ui` in your project, run the following command

**using npm:**

```bash
npm install --save @candy-doc/ui
```

**using yarn:**

```bash
yarn add --dev @candy-doc/ui
```

## Usage

Once you have installed the package, you can access the components using the `import` instruction such as:

```js
import "@candy-doc/ui/src/components/Button";
```

Once imported, use the component as HTML tag:

```js
<candy-button label="Click me" size="md"></candy-button>
```

Which will result to this rendering:

> <candy-button label="Click me" size="md"></candy-button>

## Custom style

Because of the Shadow DOM, styles of the web components are external.<br/>
If you wish to override any component style, each of them has its own [part](https://developer.mozilla.org/fr/docs/Web/CSS/::part) attribute (cf. component's doc page).

Following button example, button's part value is `button`. Here is how you can override your button:

```css
candy-button::part(button) {
  width: 6rem;
  height: 2rem;
}
```
